## Fitur: di halaman laporan/rekap-input, tambahkan fungsi export pdf

### Konteks Codebase
Aplikasi ini dibangun menggunakan Laravel dan Livewire. Halaman "rekap-input" diimplementasikan sebagai komponen Livewire (`App\Livewire\Laporan\RekapInput.php`) yang bertanggung jawab untuk mengambil data berdasarkan rentang tanggal dan menampilkannya. Terdapat pola ekspor data Excel yang sudah ada menggunakan `Maatwebsite\Excel` melalui kelas `App\Exports\RekapInputExport.php` yang merender tampilan Blade (`resources/views/exports/rekap-input.blade.php`). Pola ini akan direplikasi untuk fungsionalitas ekspor PDF.

### File yang Perlu Dibuat/Dimodifikasi
- `composer.json` — Menambahkan dependensi untuk library PDF (`barryvdh/laravel-dompdf`).
- `app/Livewire/Laporan/RekapInput.php` — Menambahkan metode `exportPdf()` untuk memicu proses ekspor PDF.
- `app/Exports/RekapInputPdf.php` — Membuat kelas ekspor PDF baru yang serupa dengan `RekapInputExport.php`, menggunakan library `dompdf` untuk merender tampilan Blade ke PDF.
- `resources/views/livewire/laporan/rekap-input.blade.php` — Menambahkan tombol atau tautan untuk memicu ekspor PDF.
- `resources/views/exports/rekap-input-pdf.blade.php` — Membuat tampilan Blade baru yang didedikasikan untuk layout dan styling laporan PDF.

### Implementation Steps
1.  **Instalasi library PDF:** Tambahkan `barryvdh/laravel-dompdf` ke `composer.json` dan jalankan `composer update`.
    ```bash
    composer require barryvdh/laravel-dompdf
    ```
2.  **Konfigurasi `dompdf`:** Publikasikan konfigurasi `dompdf` jika diperlukan dan pastikan sudah benar.
    ```bash
    php artisan vendor:publish --provider="Barryvdh\DomPDF\ServiceProvider" --tag="dompdf_config"
    ```
3.  **Buat kelas `RekapInputPdf`:** Buat file `app/Exports/RekapInputPdf.php`. Kelas ini akan bertanggung jawab untuk mengambil data yang sama dengan `RekapInputExport` dan mengoperkannya ke tampilan Blade baru untuk PDF.
    ```php
    <?php
    namespace App\Exports;

    use Illuminate\Contracts\View\View;
    use Illuminate\Support\Facades\DB;
    use App\Models\Referensi\RefPasar;
    use Barryvdh\DomPDF\Facade\Pdf; // Import PDF facade

    class RekapInputPdf
    {
        public $dateFrom;
        public $dateTo;

        public function __construct($dateFrom, $dateTo)
        {
            $this->dateFrom = $dateFrom;
            $this->dateTo = $dateTo;
        }

        public function generatePdf()
        {
            // Ambil semua tanggal dalam range yang ada datanya
            $tanggals = DB::table('t_siba_komoditas')
                ->select('detail_tgl')
                ->whereNotNull('detail_tgl')
                ->when($this->dateFrom, fn($q) => $q->where('detail_tgl', '>=', $this->dateFrom))
                ->when($this->dateTo,   fn($q) => $q->where('detail_tgl', '<=', $this->dateTo))
                ->groupBy('detail_tgl')
                ->orderBy('detail_tgl', 'asc')
                ->pluck('detail_tgl')
                ->toArray();

            // Ambil semua pasar
            $pasars = RefPasar::orderBy('namapasar', 'asc')->get();

            // Hitung jumlah data per pasar per tanggal
            $counts = DB::table('t_siba_komoditas')
                ->select('pasar_id', 'detail_tgl', DB::raw('COUNT(*) as jumlah'))
                ->whereIn('detail_tgl', $tanggals)
                ->groupBy('pasar_id', 'detail_tgl')
                ->get()
                ->groupBy('pasar_id');

            $pdf = Pdf::loadView('exports.rekap-input-pdf', [
                'tanggals' => $tanggals,
                'pasars'   => $pasars,
                'counts'   => $counts,
                'dateFrom' => $this->dateFrom,
                'dateTo'   => $this->dateTo,
            ]);

            return $pdf->download('rekap_input_harga_komoditas_' . now()->format('Ymd_His') . '.pdf');
        }
    }
    ```
4.  **Tambahkan metode `exportPdf` di `App\Livewire\Laporan\RekapInput.php`:** Metode ini akan memanggil `RekapInputPdf` yang baru dibuat.
    ```php
    // Dalam kelas RekapInput
    use App\Exports\RekapInputPdf; // Tambahkan ini di bagian atas

    public function exportPdf()
    {
        $export = new RekapInputPdf($this->dateFrom, $this->dateTo);
        return $export->generatePdf();
    }
    ```
5.  **Buat tampilan Blade untuk PDF:** Buat file `resources/views/exports/rekap-input-pdf.blade.php`. Isi file ini dengan struktur HTML dan styling yang sesuai untuk laporan PDF, menggunakan variabel `$tanggals`, `$pasars`, `$counts`, `$dateFrom`, dan `$dateTo`. Ini bisa berupa adaptasi dari `resources/views/exports/rekap-input.blade.php`.
6.  **Tambahkan tombol ekspor PDF di tampilan Livewire:** Edit `resources/views/livewire/laporan/rekap-input.blade.php` atau tampilan induknya untuk menambahkan tombol yang memanggil `wire:click="exportPdf"`.
    ```html
    <button wire:click="exportPdf" class="btn btn-danger">Export PDF</button>
    ```

### Acceptance Criteria
- [ ] Tombol "Export PDF" tersedia di halaman laporan/rekap-input.
- [ ] Mengklik tombol "Export PDF" akan mengunduh file PDF yang berisi data laporan rekap-input.
- [ ] Konten PDF mencerminkan data yang difilter oleh `dateFrom` dan `dateTo` yang dipilih di halaman.
- [ ] Format dan tata letak PDF dapat dibaca dengan baik.

### Stack Notes
- Framework: laravel
- Database: postgres
- Pola yang ditemukan: Penggunaan Livewire untuk interaktivitas frontend, `Maatwebsite\Excel` untuk ekspor Excel, dan render data melalui tampilan Blade. Database diakses melalui query builder `DB::table` dan Eloquent (`RefPasar`).
