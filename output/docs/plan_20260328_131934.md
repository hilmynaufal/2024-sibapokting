## Fitur: di halaman laporan/rekap-input, tambahkan fungsi export pdf

### Konteks Codebase
Proyek ini menggunakan Laravel dengan Livewire untuk frontend. Eksport data saat ini diimplementasikan menggunakan `Maatwebsite\Excel` dengan kelas `RekapInputExport` yang mengambil data dari database dan merender view Blade (`exports.rekap-input`) untuk pembuatan file Excel. Tidak ada library PDF yang terdeteksi di `composer.json`. Routing diatur di `routes/web.php` dan `laporan.rekap-input` diarahkan ke komponen Livewire `App\Livewire\Laporan\RekapInput`.

### File yang Perlu Dibuat/Dimodifikasi
- `composer.json` — Tambahkan dependensi `barryvdh/laravel-dompdf`.
- `config/app.php` — (Opsional, tapi sebaiknya otomatis) Daftarkan provider dan alias `Dompdf`.
- `app/Exports/RekapInputPdfExport.php` — Buat kelas baru untuk menangani logika ekspor PDF, mirip dengan `RekapInputExport.php`.
- `app/Livewire/Laporan/RekapInput.php` — Tambahkan method untuk memicu ekspor PDF.
- `resources/views/laporan/rekap-input.blade.php` atau `resources/views/exports/rekap-input.blade.php` — Tambahkan tombol/link untuk memicu ekspor PDF.
- `routes/web.php` — Tambahkan route baru untuk ekspor PDF.

### Implementation Steps
1.  **Install `barryvdh/laravel-dompdf`**:
    - Jalankan `composer require barryvdh/laravel-dompdf`.
    - (Opsional, untuk Laravel 5.5+, harusnya auto-discover) Tambahkan `Barryvdh\DomPDF\ServiceProvider::class` ke array `providers` di `config/app.php` dan `'Pdf' => Barryvdh\DomPDF\Facade::class` ke array `aliases`.

2.  **Buat kelas `RekapInputPdfExport`**:
    - Buat file `app/Exports/RekapInputPdfExport.php`.
    - Kelas ini akan mengextend `Barryvdh\DomPDF\Facade` atau menggunakannya secara langsung.
    - Struktur kelas akan mirip dengan `RekapInputExport`, namun method `view()` akan merender view `exports.rekap-input` ke format PDF.

3.  **Modifikasi `App\Livewire\Laporan\RekapInput.php`**:
    - Tambahkan sebuah method baru, misalnya `exportPdf()`, yang akan memanggil `RekapInputPdfExport` untuk menghasilkan PDF dan mengunduhnya.
    - Method ini akan menerima parameter `dateFrom` dan `dateTo` seperti halnya `RekapInputExport`.

4.  **Tambahkan Tombol/Link di View**:
    - Edit view `resources/views/laporan/rekap-input.blade.php` (atau view yang sesuai untuk halaman rekap-input).
    - Tambahkan tombol "Export PDF" yang akan memicu method `exportPdf()` di komponen Livewire. Contoh: `<button wire:click="exportPdf">Export PDF</button>`. Pastikan untuk meneruskan parameter tanggal jika ada.

5.  **Tambahkan Route untuk PDF Export**:
    - Tambahkan route baru di `routes/web.php` untuk menangani permintaan ekspor PDF, misalnya:
      ```php
      Route::get('laporan/rekap-input/pdf', [App\Livewire\Laporan\RekapInput::class, 'exportPdf'])->name('laporan.rekap-input.pdf');
      ```
      Namun, karena menggunakan Livewire, kita bisa memanggil method Livewire secara langsung atau menggunakan aksi unduh Livewire. Pendekatan `wire:click` di Livewire akan lebih langsung.

### Acceptance Criteria
- [ ] Pengguna dapat melihat tombol "Export PDF" di halaman laporan/rekap-input.
- [ ] Mengklik tombol "Export PDF" akan mengunduh file PDF yang berisi data laporan rekap input.
- [ ] Format dan konten PDF sesuai dengan tampilan laporan di halaman `exports.rekap-input` (atau tampilan yang digunakan `RekapInputExport`).
- [ ] Filter tanggal (jika diterapkan pada tampilan) juga harus diterapkan pada ekspor PDF.
