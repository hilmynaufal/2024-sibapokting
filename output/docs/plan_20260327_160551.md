## Fitur: di halaman laporan/rekap-input, tambahkan fungsi export pdf

### Konteks Codebase
The application is built with Laravel and Livewire. Reports are handled by Livewire components. The `laporan/rekap-input` page is served by `App\Livewire\Laporan\RekapInput`. Data retrieval for this report involves querying `t_siba_komoditas` and `RefPasar` models, grouped by `detail_tgl` and `pasar_id`. Existing export functionality utilizes `Maatwebsite\Excel` via `RekapInputExport` which renders an Excel-specific Blade view (`exports.rekap-input`). There is no explicit PDF generation library listed in `composer.json`'s `require` section, suggesting it needs to be added. The application uses a `components.layouts.keenthemes.page` layout.

### Implementation Steps
1.  **Install `barryvdh/laravel-dompdf`**:
    *   Run `composer require barryvdh/laravel-dompdf`.
    *   (Optional, if not auto-discovered) Add `Barryvdh\DomPDF\ServiceProvider::class` to `providers` array in `config/app.php`.
    *   (Optional, if not auto-discovered) Add `'PDF' => Barryvdh\DomPDF\Facade::class` to `aliases` array in `config/app.php`.
    *   Run `php artisan vendor:publish --provider="Barryvdh\DomPDF\ServiceProvider"` to publish config and views.
2.  **Create a new Blade view for PDF export**:
    *   Create `resources/views/exports/rekap-input-pdf.blade.php`.
    *   Copy the relevant content from `resources/views/exports/rekap-input.blade.php` (or `resources/views/livewire/laporan/rekap-input.blade.php`) into this new file. Adjust styling for PDF if necessary.
3.  **Add `exportPdf()` method to `App\Livewire\Laporan\RekapInput.php`**:
    *   This method will retrieve the same data (`tanggals`, `pasars`, `counts`, `dateFrom`, `dateTo`) as the `render()` method and the `RekapInputExport` class.
    *   It will then load the `resources/views/exports/rekap-input-pdf.blade.php` view with this data and generate the PDF using `Dompdf`.
    *   Example snippet for the method:
        ```php
        use Barryvdh\DomPDF\Facade\Pdf; // Add this use statement

        public function exportPdf()
        {
            // ... (data retrieval logic, same as render() or RekapInputExport::view())
            $tanggals = DB::table('t_siba_komoditas')
                ->select('detail_tgl')
                ->whereNotNull('detail_tgl')
                ->when($this->dateFrom, fn($q) => $q->where('detail_tgl', '>=', $this->dateFrom))
                ->when($this->dateTo,   fn($q) => $q->where('detail_tgl', '<=', $this->dateTo))
                ->groupBy('detail_tgl')
                ->orderBy('detail_tgl', 'asc')
                ->pluck('detail_tgl')
                ->toArray();

            $pasars = RefPasar::orderBy('namapasar', 'asc')->get();

            $counts = DB::table('t_siba_komoditas')
                ->select('pasar_id', 'detail_tgl', DB::raw('COUNT(*) as jumlah'))
                ->whereIn('detail_tgl', $tanggals)
                ->groupBy('pasar_id', 'detail_tgl')
                ->get()
                ->groupBy('pasar_id');

            $data = [
                'tanggals' => $tanggals,
                'pasars'   => $pasars,
                'counts'   => $counts,
                'dateFrom' => $this->dateFrom,
                'dateTo'   => $this->dateTo,
            ];

            $pdf = Pdf::loadView('exports.rekap-input-pdf', $data);
            return response()->streamDownload(function() use ($pdf) {
                echo $pdf->output();
            }, 'rekap_input_harga_komoditas_' . Carbon::now()->format('Ymd_His') . '.pdf');
        }
        ```
4.  **Add PDF export button to `resources/views/livewire/laporan/rekap-input.blade.php`**:
    *   Locate the existing Excel export button.
    *   Add a new button next to it that calls `Livewire.dispatch('exportPdf')` or `wire:click="exportPdf"`.
